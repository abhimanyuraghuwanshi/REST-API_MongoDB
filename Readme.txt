This Restfull-API project is build on ExpressJS and have MongoDB as database which has a complete demonstration of real time CRUD operation with respect to user:
1 user registration
2 fetch user details
3 updating user profile
4 deleting user
5 login with JWT

The information related to API is present in file name API-handler.txt inside mywork folder.  


inorder to run project 
1. In cmd run npm init.
2. run npm start.
